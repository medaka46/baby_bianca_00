How it works

Each Thai phrase is looked up in a SQLite glossary first; cache misses fall through to a heuristic that transliterates personal names (Royal Thai General System), or to Argos Translate (offline) for general Thai. New API translations are written back to the glossary so the same phrase is free next time. Names and abbreviations are classified separately so they can be hand-curated.
